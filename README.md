# Smart Student Task Scheduler

This is a simple and intelligent web-based application that helps students organize their daily tasks and automatically generate a timetable.

## 🚀 Features
- Add multiple tasks easily
- Automatically generate a daily timetable
- Assigns time slots to each task
- Simple and user-friendly interface
- Dynamic task scheduling using JavaScript

## 🧠 How it Works
The user enters tasks, and the system automatically assigns each task a time slot starting from 9:00 AM. It generates a structured timetable based on the number of tasks entered.

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in any browser
3. Add tasks and click "Generate Timetable"

## 📌 Example
If you enter 3 tasks:
- Study
- Assignment
- Practice

The system will generate:
- 9:00 - 10:00 → Study  
- 10:00 - 11:00 → Assignment  
- 11:00 - 12:00 → Practice
